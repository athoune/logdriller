Log driller
===========

Parse and boost your apache log, put them in an Influxdb, and query them.


Install
-------

Install geoip lib, and Influxdb

    virtualenv .
    source bin/activate
    pip install requirements.txt

Fetch a fresh [geoip database](http://dev.maxmind.com/geoip/geoip2/geolite2/)
